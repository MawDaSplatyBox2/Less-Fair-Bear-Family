# Less-Fair-Bear-Family
This add-on mod for the [Bear Family](https://www.nexusmods.com/stardewvalley/mods/16197) rebalances the prices at Gunnar's shop and the Bears' vending machine, making things more expensive by default and providing config options to change the price scaling. Sell values have been adjusted so that items can't be easily resold for a profit.
